🔄 Data Flow

How It All Works

🎮 Gameplay Flow
1 Player loads game via CloudFront CDN
2 S3 serves HTML/CSS/JS files
3 JavaScript generates math questions locally
4 Player answers questions, score computed
5 Score POSTed to API Gateway endpoint
6 Lambda writes score to DynamoDB table
🏆 Leaderboard Flow
1 Player opens Leaderboard page
2 JavaScript calls GET /leaderboard API
3 API Gateway triggers Lambda function
4 Lambda queries DynamoDB (Global Secondary Index)
5 Top 100 scores returned as JSON
6 Browser renders animated leaderboard
🔐 Auth Flow
1 User submits register/login form
2 Cognito SDK called from browser
3 Cognito validates credentials
4 JWT Access + ID Tokens returned
5 Token stored in localStorage
6 Token sent in Authorization header for all API calls
